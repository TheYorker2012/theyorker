CRUD

CRUD (acronym of Create Read Update Delete) is a common task of a CMS (acronym of Content Management System).
Rapyd is thought to build CMS, and to build a CRUD is easy, you need to join two or three components.

Note: CI scaffolding library is a CRUD utility, but it is not fully customizable, and can be used as generic DB editing utility, not to implement custom editing interfaces.
But, rapyd way to build crud (DF+DG+DE), "do not" sniff table fields, so it's more complex.
