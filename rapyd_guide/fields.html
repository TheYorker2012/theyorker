Fields

Fields are a set of classes that can be used in DataForm (and extended DataEdit & DataFilter).

Fiels are a collection of classes to manage single items of a form

  • input
  • password
  • dropdown
  • textarea
  • checkbox
  • datetime
  • editor
  • autoupdate
  • submit, reset & button
  • upload
  • free
  • container
  • iframe
  • colorpicker
  • html

inputField

An Input.

params: string $label, string $fieldname.
properties: insertValue, updateValue, mode, rule, group.
methods: build(), when($status).

passwordField

An Input, where type = password, with possibility to encrypt data, hide value in view state, and empty value in modify state.
contributed by Nick Crossland.
params: string $label, string $fieldname.
properties: insertValue, updateValue, mode, rule, group.
encrypt (bool),
show_null, show_empty, show_mask_encrypted, show_mask_hidden (strings)
methods: build(), when($status).

dropdownField

A Select.

params: string $label, string $fieldname, [$values_arr].
properties: onchange, insertValue, updateValue, mode, rule, group.
methods: option($value, $label), append($extra_text), build(), when($status).

textareaField

A Textarea.

params: string $label, string $fieldname.
properties: cols, rows, insertValue, updateValue, mode, rule, group.
methods: build(), when($status).

checkboxField

A Checkbox.

params: string $label, string $fieldname, mixed $checkedValue , mixed $uncheckedValue.
properties: insertValue, updateValue, mode, rule, group.
methods: build(), when($status).

dateField

A Date-Picker DHTML calendar based on jscalendar (rapyd 3th party library).

params: string $label, string $fieldname, string $format ("eu" for european timestamp: dd/mm/yyyy, or "us" for US format: mm/dd/yyyy)
properties: insertValue, updateValue, mode, rule, group.
methods: build(), when($status).

editorField

A wysiwyg DHTML editor (like a small Word application), derived from TinyMCE, and TinyFCK

params: string $label, string $fieldname, string $format ("eu" for european timestamp: dd/mm/yyyy, or "us" for US format: mm/dd/yyyy)
properties: insertValue, updateValue, mode, rule, group.
methods: build(), when($status).

autoUpdateField

Not properly a hidden field, it can store & update "code-passed" values (instead of POSTs).
In the sample it is used to store (widhout output) a datetime, "last record update"

params: string $fieldname, mixed $insertValue , mixed $updateValue.
properties: mode, rule, group.
methods: build(), when($status).

resetField submitField & buttonField

Buttons, to exec actions.

params: string $fieldname, mixed $insertValue , mixed $updateValue.
properties: group.
methods: build(), when($status).

uploadField

A Input (file type), to upload files. It is based upon CI Upload library.
In the sample, the "thumb" property is used to make a resized thumbnail (max 63x91 pixels), using CI image_lib (image manipulation class)

params: string $label, string $fieldname
properties: cols, rows, insertValue, updateValue, mode, rule, group
methods: build(), when($status).

freeField

A empty field, it can contain a passed html string, basic scope is to enable, inside forms (DataForm/DataEdit) a custom area to display complex/custom fields.

params: string $uniqueName, string $html_to_print
properties: insertValue, updateValue, mode, rule, group
methods: build(), when($status).

containerField

A div, it can contain a passed html string, basic scope is to enable, inside forms (DataForm/DataEdit) a custom area to display messages, custom and static fields.. etc.

params: string $uniqueName, string $html_to_print
properties: insertValue, updateValue, mode, rule, group
methods: build(), when($status).

iframeField

An iframe, basic scope is to enable master-detail editing (..see advenced tips).

params: string $uniqueName, string $uri, string $iframe_height (pixels)
properties: insertValue, updateValue, mode, rule, group
methods: build(), when($status).

colorpickerField

A Color-Picker DHTML, based on colorpicker (rapyd 3th party library).

params: string $label, string $fieldname
properties: cols, rows, insertValue, updateValue, mode, rule, group
methods: build(), when($status).

htmlField

Same of Textarea but in "show" status it has a better output layout, and in "create/modify" status a smaller font.

params: string $label, string $fieldname
properties: sames of textarea.
methods: build(), when($status).
