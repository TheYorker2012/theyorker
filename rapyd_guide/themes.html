Themes / Templates

The theme system of rapyd is thought to standardize components layout.
In the /system/application/rapyd/view there is a "default" folder that is the the default theme views container.

Each (template-based) component has a view, so you can manage your changes or (better) build your custom theme (by copying and renaming the default folder theme)

in the rapyd config file /system/application/config/rapyd.php you can set your theme folder.
