DataSet

DataSet is a "support" class for an easy pagination, it has all pagination $config "derived" from CI pagination class as parameter (for example $dataset->uri_segment) and it work with a given data-source

Sample:

constructor:

DataSet($datasource)
param: mixed $datasource

main properties:

base_url
(the same of CI pagination config) used to build nav-links.
It's optional, can be autodetected, if last uri_segment is number it assume that is the page-offset.

uri_segment
(the same of CI pagination config) used to build nav-links
It's optional, can be autodetected (if not specified rapyd assume it is the last uri_segment).

per_page
(the same of CI pagination config) max number of record per page

.. and all other pagination $congif["vars"]

main methods:

build()
exec queries if datasource is ActiveRecord, build paged array (subset of results for current page) and the html of navigator
It fill $dataset->data and $dataset->navigator

output:

data
array, "subset" of record for current page, you need to iterate this in the view page.

navigator
string, html navigator, you can customize the nav layout (see CI user guide, and set correct properties before build).
